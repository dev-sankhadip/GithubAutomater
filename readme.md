## configure

### For linux user

* Download cli.py file
* install click python module globally in your system
```sh
pip3 install -u click
 ```
 * make cli.py file executable
 ```sh
 chmod +x cli.py
 ```
 * move cli.py file to bin folder
 ```sh
 mv cli.py /bin
 ```
